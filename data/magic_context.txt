Cuenta el cuento como si fueras una niña o un niño.

En general:
El texto puede ser una historia, una descripción, una noticia, una receta o una poesía.
Empieza a lo grande: Acción, suspense y final satisfactorio.
El texto debe ser, ante todo, educativo.
Alineado con el currículo escolar.
Curiosidad.
Mejor hablar en primera persona, pero no siempre.
Refleja la diversidad.
El héroe o la heroína debe ser, más o menos, de la edad del lector o lectora.


El texto necesita una causa que haga avanzar la acción.

En particular:

Si el relato es para 1º o 2º de primaria:
Se busca enseñar a leer.
Alfabetización: letras y sonidos.
El relato como un juguete.
Conciencia fonológica
literalidad
El texto debe apelar a las emociones.


Si el relato es para 3º o 4º de primaria:
Se busca enriquecer el vocabulario del lector.
Se puede empezar con preguntas.
Diálogos para las partes más intensas y muy cortos.
Inferencias
comprensión global
El texto debe apelar a las emociones.


Si el relato es para 5º o 6º de primaria:
Diálogos para las partes más intensas y muy cortos.
Se puede empezar con preguntas.
Se busca inferencia. Que haya partes no dichas que se tengan que interpretar.
Se busca el análisis, la reflexión y la lectura crítica.
